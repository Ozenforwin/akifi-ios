# /sprint — Sprint Planning

## Контекст
$ARGUMENTS

## Шаги

1. Прочитай текущий контекст из vault:
   - `~/Documents/agent-factory-brain/Hot/hot.md`
   - `~/Documents/agent-factory-brain/Hot/current-sprint.md`
   - `~/Documents/agent-factory-brain/PRD/_index.md`

2. Вызови **@product-manager** — приоритизируй бэклог, определи P0/P1/P2

3. Вызови **@team-lead** — декомпозируй задачи, оцени ёмкость, составь план:
   - P0 — Must Have
   - P1 — Should Have
   - P2 — Stretch

4. Создай файл спринта используя шаблон:
   `~/Documents/agent-factory-brain/Templates/sprint-template.md`

5. Обнови `Hot/current-sprint.md` и `Hot/hot.md`
